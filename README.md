# はせがわ見守りプラットフォーム

NILM技術による認知症早期検出と生活見守りサービスのプロトタイプ

## 🎯 デモURL

**スマホフレーム付きデモ:** [demo.html](demo.html)  
**フルスクリーンアプリ:** [index.html](index.html)

## 📱 主要機能

### 1. ダッシュボード
- 朝の3動作スコア（トイレ・ケトル・洗濯機）
- 認知症スコア表示（0-100）
- リアルタイム健康状態モニタリング

### 2. 家電モニタリング
- 5つの優先家電監視（洗濯機・電子レンジ・電気ケトル・温水洗浄便座・玄関照明）
- 24時間活動パターングラフ
- 使用状況統計

### 3. 通知・アラート
- 認知症兆候の早期検出
- 繰り返し行動警告
- 夜間活動異常通知

### 4. 生活リズムレポート
- 週次スコアトレンド
- 家電使用頻度分析
- AIによるインサイトと推奨事項

## 📦 デプロイ方法

### GitHub Pagesへのデプロイ

1. GitHubリポジトリを作成
2. このフォルダの内容をリポジトリにプッシュ
3. Settings → Pages → Source を "main branch" に設定
4. `https://your-username.github.io/repository-name/demo.html` でアクセス

### Netlify/Vercelへのデプロイ

1. このフォルダをドラッグ&ドロップ
2. 自動的にデプロイ完了

### 独自サーバーへのデプロイ

```bash
# このフォルダをWebサーバーのドキュメントルートにコピー
cp -r hasegawa_mimamori_web/* /var/www/html/
```

## 🚀 ローカルでの実行

```bash
# Pythonの簡易サーバーで実行
python3 -m http.server 8000

# ブラウザで開く
# http://localhost:8000/demo.html
```
